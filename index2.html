WHY THIS MATTERS

ANNs compute every neuron on every clock tick — 100% active, always. SNNs only activate when a spike arrives.

Sparsity = how often events actually happen. At 30% sparsity, 70% of the SNN is doing nothing — spending zero energy. An ANN can't do this.

THE LEARNING RULE

If A fires before B → A probably caused B to fire → synapse strengthens.

If B fires before A → A had nothing to do with it → synapse weakens.

No backward pass. No global error signal. Just two neurons and their spike timestamps. This is how a brain learns without a teacher.

VS BACKPROPAGATION
ANN backprop: run the whole network, calculate global error, send it backwards through every layer, update every weight. Requires the full dataset to be present.
STDP: each synapse updates itself using only local timing info. Continuous, online, no dataset needed.
